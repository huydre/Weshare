### :gear: Installation

* Install Back-end
```
cd weshare-backend
```
```
docker-compose exec backend  sh
```
```
python manage.py migrate
```

* Install Front-end
```
cd weshare-frontend
```
```
npm i
```
```
npm run dev
```
